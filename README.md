# Monkey Lang

Development of the Monkey programming language with [zig](https://www.ziglang.org/) based of *Writing an Interpreter with Go by Thorsten Ball*

## Components:
    + [x] Lexer
    + [x] Parser:
        + [x] Identifiers
        + [x] Integer Literals
        + [x] Boolean Literals
        + [x] Function Literals
        + [x] Assignment statements
        + [x] Return statements
        + [x] Expressions:
            + [x] Prefix expressions
            + [x] Infix expressions
            + [x] Grouped expressions
            + [x] If expressions
            + [x] Grouped expressions
            + [x] Call expressions
